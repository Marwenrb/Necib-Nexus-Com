## üèÅ Getting Started

First, install the dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üöÄ Deployment

Prepare for production deployment:

```bash
pnpm build
```

This will:
1. Optimize all images in the public directory
2. Generate WebP versions of images
3. Build the production-ready application with all optimizations
4. Create a sitemap for SEO

The optimized build can be tested with:

```bash
pnpm start
```

## üîß Environment Setup

Create a `.env.local` file with the following variables:

```
NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAILJS_USER_ID=your_user_id
NEXT_PUBLIC_EMAILJS_CLUB_TEMPLATE_ID=your_club_template_id
NEXT_PUBLIC_MAPBOX_TOKEN=your_mapbox_token
```
